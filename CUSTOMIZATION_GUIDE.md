## üéØ Quick Start Guide

### Step 1: Open Customization Panel
1. Go to Invoice Generator page
2. Click "Customize Design" button in header
3. Panel opens on the left side

### Step 2: Choose Your Base
1. Select a template (Custom, Professional, Modern, or Minimal)
2. Custom template gives you full control
3. Other templates have fixed colors but customizable layout

### Step 3: Apply a Color Preset (Custom Template Only)
1. In "Color Scheme" section
2. Click one of the 6 preset buttons
3. Or customize each color individually

### Step 4: Position Your Logo
1. Upload logo in the form
2. In "Logo Settings" section
3. Choose placement (top-left, center, right, watermark)
4. Adjust size and opacity

### Step 5: Fine-Tune Layout
1. Adjust padding for page margins
2. Adjust section spacing for breathing room
3. Toggle borders and adjust roundness

### Step 6: Hide Settings
1. Click "Hide Settings" to see full preview
2. Toggle back anytime to make changes

## üí° Tips & Best Practices

### For Professional Look
- Use **Professional** or **Custom** template
- Choose **Blue** or **Slate** color preset
- Keep logo at **top-left** or **top-center**
- Use **moderate spacing** (24px sections, 32px padding)
- Enable **borders** with **8px radius**

### For Creative/Modern Look
- Use **Modern** or **Custom** template
- Choose **Rose**, **Emerald**, or **Amber** preset
- Try **watermark** logo placement
- Use **generous spacing** (32px sections, 48px padding)
- Experiment with **gradient** header style

### For Minimalist Look
- Use **Minimal** template
- Keep **default colors**
- Use **top-left** logo placement
- Use **tight spacing** (16px sections, 24px padding)
- Disable **borders**

### For Maximum Readability
- Choose **high contrast** colors
- Use **14px body** text minimum
- Enable **table borders**
- Use **striped rows** in tables
- Keep **adequate padding** (32px+)

## ‚ùì Troubleshooting

### Customization Not Saving
- Settings are saved to browser localStorage
- Clear cache may reset settings
- Use same browser to maintain settings

### Logo Not Displaying
- Check file format (PNG, JPG, SVG supported)
- Ensure file size < 5MB
- Try different placement option

### Scrollbar Not Visible
- Scrollbar appears only when content overflows
- Try adding more line items or longer descriptions
- Check browser compatibility (works best in Chrome, Edge, Safari)
